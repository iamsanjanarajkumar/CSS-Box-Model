CSS BOX MODEL...!!

Every element that can be displayed on a web page is comprised of one or more rectangular boxes .CSS box model typically describes how these rectangular boxes are laid out on a web page. these boxes can have different properties and can interact with each other in different ways , but every box has a content area and optional surrounding padding, border& margin area.
